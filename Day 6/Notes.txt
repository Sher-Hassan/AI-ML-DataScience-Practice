## Numpy:

NumPy is a fundamental library for scientific computing in Python. It provides support for arrays and matrices, along with a collection of mathematical functions to operate on these data structures. In this lesson, we will cover the basics of NumPy, focusing on arrays and vectorized operations.

# Numpy Notes
import numpy as np

## create array using numpy
## create a 1d array
arr1=np.array([1,2,3,4,5])
print(arr1)
# this is not a list
print(type(arr1))
# we can also see the shape of array
print(arr1.shape)
# in 1d array there are no rows and columns
# reshaping 1d array to 2d array
arr2 = np.array([1,2,3,4,5])
arr2.reshape(1,5) # One row and five columns
# Now it will respond as array([[1, 2, 3, 4, 5]]) which has two [[]] brackets which shows it is 2d array. similarly 3d will have three brackets

# similarly we can use two [[]] for creating 2d and [[[]]] for creating 3d array
arr3 = np.array([[1,2,3,4,5]])
print(arr3)

# for multiple rows
arr4 = np.array([[1,2,3,4], [5,6,7,8]])
print(arr4)
print(arr4.shape) # (2,4) cuz there are two rows and four columns

# Creating arrays with inbuilt functions
np.arange(0,10,2) # range from 0 to 10 with step 2
# We can use multiple func like this
np.arange(0,10,2).reshape(5,1)

# Inbuilt functions for Numpy
np.ones((3,4)) # make an array with rows,columns consisting 1 as element

np.eye(3) # Creates an identiny matrix with argument as number of diagonal elements

# Attributes of Numpy
arr = np.array([[1, 2, 3], [4, 5, 6]])

print("Array:\n", arr)
print("Shape:", arr.shape)  # Output: (2, 3)
print("Number of dimensions:", arr.ndim)  # Output: 2
print("Size (number of elements):", arr.size)  # Output: 6
print("Data type:", arr.dtype)  # Output: int64 (may vary based on platform)
print("Item size (in bytes):", arr.itemsize)  # Output: 8 (may vary based on platform)

## Numpy Vectorized Operation
arr1 = np.array([1,2,3,4,5])
arr2 = np.array([10,20,30,40,50])

# Element Wise addition
print("Addition:", arr1+arr2)

# Elememt Wise Subtraction
print("Subtraction: ", arr2 - arr1)

# similarly is multiplication, division etc

# Pandas: Data Manipulation Library
Pandas is a powerful data manipulation library in Python, widely used for data analysis and data cleaning. It provides two primary data structures:
- **Series**: One-dimensional array-like object
- **DataFrame**: Two-dimensional, size-mutable, tabular data structure with labeled axes (rows and columns)

## Importing Pandas
import pandas as pd

## Pandas Series
A Pandas Series is a one-dimensional array-like object that can hold any data type. It is similar to a column in a table.

Example:
data = [1,2,3,4,5]
series = pd.Series(data)
print("Series \n", series)
print(type(series))
# Output:
# 0    1
# 1    2
# 2    3
# 3    4
# 4    5
# dtype: int64
# <class 'pandas.Series'>

### Creating Series from Dictionary

data = {'a':1, 'b':2, 'c':3}
series_dict = pd.Series(data)
print(series_dict)
# Keys become index, values are series values
# Output:
# a    1
# b    2
# c    3
# dtype: int64

# Custom index

data = [10,20,30]
index = ['a','b','c']
pd.Series(data, index=index)
# Output:
# a    10
# b    20
# c    30
# dtype: int64

## Pandas DataFrame
A Pandas DataFrame is a two-dimensional, size-mutable, tabular data structure with labeled axes (rows and columns).

### Reading Data
- CSV & Text: pd.read_csv() for comma-separated values or general delimited text files.
- Excel: pd.read_excel() for .xlsx and .xls files (supports multiple sheets).
- JSON: pd.read_json() for web-based data structures.
- SQL Databases: pd.read_sql() to pull data directly from databases like PostgreSQL, MySQL, or SQLite.

### Creating DataFrame from Dictionary of Lists

data = {
    'Name': ['Sher', 'Hassan', 'Khan'],
    'Age': [18,19,20],
    'City': ['Quetta', 'Islamabad', 'Zhob']
}
df = pd.DataFrame(data)
