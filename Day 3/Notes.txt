- To activate an environment
- conda activate venv/  (this venv will be activated) (remember to run in command prompt in terminal)
- Further installation of packages will take place in this environment
- conda deactivate  (To deactivate)
- Now to install the packages inside requirements.txt:
- pip install -r requirements.txt (These will be installed in the environment created)

How to create your own custom module/package (can be called in other files)
- make a folder (ex.  package)
- first we make a file "__init__.py"  in it(specialized file used in python to define packages and their namespaces)
- remember that if we want to initialize a folder for packages (user defined) we have to make an empty file __init__.py inside it
- make a file for module/package  package_name.py in that folder.
- define its functions inside it.
- import in any file using:
from package.maths import addition
addition(2,3)
OR 
from package import maths
maths.addition(2,3)
syntax: 
from folder_name.package_name import function_name
function_name()
- By this way we can have multiple self defined functions which can be imported
- if we want to make subpackages folder inside the package folder we have to put "__init__.py" file in that too.
- we can call that too using imports
from folder_name.subpackageFolder_name.packagename import function_name
funtion_name()

    print(f"'{path}' is a file")
elif os.path.isdir(path):

else:
    print(f"'{path}' is a neither a directory nor a file")
